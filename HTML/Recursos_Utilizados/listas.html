Tipos de Listas em HTML

Listas Ordenadas

As listas ordenadas usam números para listar os itens:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
  4. Quarto item

Listas Não Ordenadas

As listas não ordenadas usam marcadores para listar os itens:

  • Maçã
  • Banana
  • Laranja

Listas de Definição

As listas de definição são usadas para apresentar termos e suas descrições:

HTML
Linguagem de Marcação de Hipertexto usada para criar páginas na Web.
CSS
Folhas de Estilo em Cascata, usadas para estilizar a aparência de elementos HTML.
JavaScript
Linguagem de programação utilizada para tornar as páginas web interativas.
